Contacts page

Another page we want to add to the site is "/contacts". We create an HTML template for it in the file "book/templates/book/contacts.html".

There is no need to pass any additional data to it, just create a class that will serve this page on the site.

By default, Django is looking for templates in all of your " <application_name>/templates" directories, so we do not include " book/templates" in the template path, because Django will do it for us.
